# Solar System Simulation (N-Body Gravity) 

A real-time solar system simulation built in Python using Pygame. The project models planetary motion using Newtonian gravity and an N-body physics system with numerical integration.

---

Features
- Newtonian gravity simulation
- N-body interactions between celestial bodies
- Real-time orbital motion
- Adjustable masses, distances, and velocities (through code)
- Verlet-based position updating
- Testing Git commands


---

Physics Concepts Used

- Newton’s Law of Universal Gravitation
- Classical mechanics
- Verlet Integration
- Basic Circular Motion Physics

---

Built With

- Python
- Pygame
- Math module
